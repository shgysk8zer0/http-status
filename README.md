## Installation

### NPM

```bash
npm i @shgysk8zer0/http-status
```

## Usage

### ES Module

```js
import { status } from '@shgysk8zer0/http-status';
// or
import { status } from 'https://unpkg.com/@shgysk8zer0@0.0.1/http-status.js';
console.log(status.OK); // 200
```
